# Smart Attendance PWA

QR-based attendance system built as a Progressive Web App (PWA).

## Quick start

1. Create `.env.local` from `.env.local.example` and fill values.
2. Install dependencies:
   - `npm install`
3. Run dev server:
   - `npm run dev`

## Notes
- This repo is a scaffold (structure + placeholders) to help you start fast.
- Docs folder is intentionally omitted as requested.
